## Install

1. Install the [selenium chrome driver](https://chromedriver.storage.googleapis.com/index.html?path=2.42/) and save it in `~/Downloads`.
2. Do `unzip chromedriver_mac64.zip`.
3. `pip install -r requirements.txt`

## Run

```
bash run.sh
```
